Computer Programming and Database Systems Certificate

It seems that you're a few courses away from getting this certificate.
You have to choose between 3 Electives from these courses:

  • INT-2030 Programming in Visual Basic
  • INT-2130 Programming in C++
  • INT-2200 Programming in Java
  • INT-2081 introduction to Oracle
  • INT-2082 Advanced Oracle and PL/SQL
  • INT-2721 Linux Operating System 1

This certificate provides an introduction to programming concepts, programming languages, database management systems and database query languages.
